# Database

A retro-styled, interactive company database and accounting dashboard with:

- User and admin login experiences
- Financial record management (revenue/expense)
- Admin configuration panel for system settings
- Live KPI cards and filtering for ease of use

## Run locally

Open `/home/runner/work/Database/Database/index.html` in a browser.

## Demo credentials

- User: `user` / `user123`
- Admin: `admin` / `admin123`

## Notes

This project is a frontend demo using local browser storage (`localStorage`) for data and settings persistence.
